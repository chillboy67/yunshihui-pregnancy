# 孕食汇 - 护理学院孕期管理系统

基于 uni-app 的孕期营养管理系统，包含移动端前端和 Python 后端。

## 项目结构

```
├── ysh/
│   ├── hl-ysh/       # uni-app 移动端前端 (Vue)
│   └── main.py       # Python 后端服务
├── mongodb/          # MongoDB 相关
├── 孕食汇说明文档.docx  # 说明文档
├── 孕食汇反馈意见.docx  # 反馈意见
└── 修改报告.docx       # 修改报告
```

## 功能

- 孕期营养分析与建议
- 饮食记录与历史
- 健康教育
- 用户反馈
